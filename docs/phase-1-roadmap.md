# Phase 1 Roadmap

## Sprint 1

- Build the analyst workbench UI
- Define the case, source, pack, and evidence models
- Add a mock engine to validate workflow language

## Sprint 2

- Add persistent storage
- Add reviewer queue and approval states
- Add exportable evidence pack downloads

## Sprint 3

- Replace the mock engine with a real workflow-service adapter
- Connect Griffin pilot pack
- Connect Cadence pilot pack

## Sprint 4

- Add SSO and role-based access
- Add source certification workflow
- Add daily pack health checks

## Sprint 5

- Add central audit log shipping
- Add case search and analytics
- Add operating dashboards for support and compliance leads
